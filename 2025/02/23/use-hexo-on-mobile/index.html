注意

Git用来干嘛的呢？

Git是用来管理Git仓库的（如推送、克隆仓库等）开源的分布式版本控制系统，本来是用在Linux开发的上的，后来在Windows上可以模拟Linux终端的环境（其实在安装git后，不需要打开Open Git Bash here，使用管理员权限下的cmd也可做到相同功能，就是搞类似ssh-keygen -t的Linux指令会比较麻烦）
